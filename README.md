Timex.Emu
========

Timex.Emu is an emulator for a popular Timex 2048 machine.

Folder structure:

docs:
  - Z80 CPU Manual
  - Z80 CPU Peripherals
  - Complete ROM

python:
  - timex.py - main entry to the emulator

rom:
  - Binary file containing ROM of actual machine

tests:
  - a suite of unit tests
